# Overview

Semble is a social knowledge network for researchers built on the AT Protocol. Think of it as [Are.na](https://www.are.na/) combined with Goodreads for science, reimagined for the decentralized web. Semble lets you curate links, organize collections, discover what your network is reading, and share insights — all while keeping ownership of your data.

🌱 _Semble is currently in alpha. Features and APIs may change as we refine the app based on feedback from users like you._
